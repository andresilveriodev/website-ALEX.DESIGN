# ALEX.DESIGN — Portfolio

Site estático do portfólio (Product Designer).

## Estrutura do projeto

```
├── index.html          # Página principal
├── favicon.svg         # Ícone do site (átomo)
├── README.md
├── templates/
│   └── design-system.html  # Documentação do design system
├── css/
│   └── main.css        # Estilos (Inter, etc.)
├── js/
│   ├── main.js        # Scripts principais (parallax, etc.)
│   └── iconify-icon.js # Ícones Iconify
├── images/            # Imagens (webp, jpg)
└── fonts/             # Fontes Inter (woff2)
```

## Como rodar

Abra `index.html` no navegador ou use um servidor local (ex.: Live Server).

## Tecnologias

- HTML5, CSS3 (Inter via local)
- Iconify (ícones)
- JavaScript vanilla (parallax, animações)
